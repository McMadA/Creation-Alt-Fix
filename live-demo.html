De Geboorte van een Website

Scroll naar beneden om te zien hoe een simpele tekst transformeert in een interactieve webpagina.

Terug naar de hoofdpagina
1

De Blauwdruk: Pure HTML

Alles begint met HTML (HyperText Markup Language). Dit is het skelet van de pagina. Het definieert de structuur en de inhoud, zoals koppen, paragrafen en knoppen, maar zonder enige opmaak.

<h1>Mijn Product</h1>
<p>Dit is een geweldig product.</p>
<button>Koop nu</button>

Mijn Product

Dit is een geweldig product dat je leven zal veranderen. Gemaakt met de beste materialen en ontworpen voor duurzaamheid.

2

De Eerste Verflaag: Basis CSS

Met CSS (Cascading Style Sheets) voegen we stijl toe. We beginnen met de basis: lettertypes, kleuren en wat opvulling (padding) om de tekst leesbaarder te maken en het er netter uit te laten zien.

body {
  font-family: 'Inter', sans-serif;
  background-color: #f0f0f0;
}
.preview-box {
  padding: 20px;
}

3

Design & Layout: Meer CSS

Nu maken we het visueel aantrekkelijk. We centreren de inhoud, geven de "box" een mooie schaduw en stijlen de knop zodat hij uitnodigend is om op te klikken. Dit is waar het design echt tot leven komt.

.preview-box {
  background: white;
  border-radius: 8px;
  box-shadow: 0 4px 15px rgba(0,0,0,0.1);
}
button {
  background-color: #007AFF;
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
}

4

De Magie: JavaScript

JavaScript voegt interactiviteit toe. Het is het brein van de pagina. Hier maken we de knop functioneel. Klik op de "Koop nu" knop in het voorbeeld om te zien wat er gebeurt!

const button = document.getElementById('js-button');
const text = document.getElementById('js-text');

button.addEventListener('click', () => {
  text.textContent = 'Bedankt voor uw aankoop!';
});
